function parseBlock(blockType,
blockText)

Block data must be passed through this to replace any chars that can break functionality with encodings.

spacesajax urls
&nbsp;ajax delimiters
&ajax delimiters
”ajax strings
’ajax strings
,array delimiters
+interpreted as spaces in urls
<br>maintaini new lines

Parameters

blockTypestring, the block’s type
blockTextstring, the block’s data

Returns

successstring, the parsed block’s data.
